# Capitech Website

Welcome to the Capitech website project! This project is designed to provide technical services for iPhones, iPads, MacBooks, and iMacs, featuring a user-friendly interface with a Japanese kawaii theme.

## Project Structure

The project is organized as follows:

```
capitech-website
├── src
│   ├── index.html          # Main HTML file for the website
│   ├── css
│   │   ├── styles.css      # Main styles for layout and typography
│   │   └── theme.css       # Kawaii theme styles
│   ├── js
│   │   ├── main.js         # Main JavaScript functionality
│   │   └── navigation.js    # Navigation functionality
│   ├── pages
│   │   ├── promotions.html  # Promotions page
│   │   ├── battery-upgrades.html # Battery upgrades page
│   │   ├── services.html    # Services offered by Capitech
│   │   └── contact.html     # Contact form and information
│   └── assets
│       └── fonts           # Custom fonts
├── package.json            # npm configuration file
└── README.md               # Project documentation
```

## Features

- **Promotions**: Stay updated with the latest offers and promotions on our promotions page.
- **Battery Upgrades**: Easily select your iPhone model and upgrade your battery capacity.
- **Technical Services**: Explore the various technical services we offer, including repairs and upgrades.
- **Contact Us**: Reach out to us through our contact form for any inquiries or support.

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd capitech-website
   ```

3. Install dependencies:
   ```
   npm install
   ```

4. Open `src/index.html` in your web browser to view the website.
